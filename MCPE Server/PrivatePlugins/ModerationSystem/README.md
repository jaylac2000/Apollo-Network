# ModerationSystem

Note: This plugin is a very highly modified version of "cucumber" (https://poggit.pmmp.io/p/cucumber)

features:

- MySQL for cross server
- Ban System
- Mute System
- Warn System
- Nick System

commands:

- /globalmute
- /ip
- /alert
- /rawtell
- /hide
- /join
- /leave
- /modexport
- /modimport
- /whois
