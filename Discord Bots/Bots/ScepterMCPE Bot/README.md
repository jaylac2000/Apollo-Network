# Scepter MCPE Bot

Bot Type: MCPE

Language:

Prefix: &

----------

General Commands:

- &help > help menu
- &ip > MCPE IP
- &shop > MCPE Shop
- &suggest > make a MCPE suggestion

Ticket Commands:

- &new > open a MCPE ticket
- &add > add someone to the support ticket
- &remove > remove someone from the support ticket
- &adminonly > make the ticket admin view only
