# Ark Bot

Bot Type: Ark

Language:

Prefix: %

----------

General Commands:

- %help > Ark help menu
- %ip > Ark IP
- %shop > Ark Shop
- %language > what language is this bot written in

Ticket Commands:

- %new > open a Ark ticket
- %add > add someone to the support ticket
- %remove > remove someone from the support ticket
- %adminonly > make the ticket admin view only
